Getting Started with MedImage

Loading images and annotating them

From the Image I/O Controls Box located on the right side, navigate to the directory your images are located and double click on an image. Next, click the "Load Canvas" button. The image will now be loaded into the canvas.

Next, you can click on the specific tools / filters you want to use to annotate the image.

  • Tool Type:
    • Pen:
        This will let you use a pen to draw or mark red lines on top of the loaded image.
    • Arrow:
        This will let you stamp a red cross arrow, circling important location on the loaded image.
  • Filters:
      Adjust the "Amount" input box to specify how much sharpen is needed.
  • Edge Detect:
      This feature will make all edges become more visible.
  • Threshold:
      This feature will adjust the loaded image base on a grayscale range.
  • Multiply RGB:
      This feature will addjust the each of the color (Red, Green, Blue) of the loaded image base on the values input into the boxes.
  • Quantize:
      Theis feature will simplify the colors of the image based on the number of bins given.
  • Grayscale:
      This feature will set the saturation of the image to become black and white.
  • Saving image:
      After you are done editing an image, in the image I/O Controls box, you will see an "Image" text box where you can insert text. Specify the new name of the image you want to save and press enter, then click "Save Canvas".
